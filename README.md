# SNAKE-WATER-GUN-ARENA
A modern web-based Snake Water Gun built from my original Python game logic and transformed into an interactive web application.

# ABOUT THE PROJECT
The Snake Water Gun core game logic was completely written by me in Python. The objective of this project was to practice programming concepts such as functions, conditionals, loops, and dictionaries. As I continued learning, I tried to enhance the user experience and explore modern development workflows, therefore I used AI-assisted tools to help transform the project into a web application. The project demonstrates my understanding of programming fundamentals and my ability to effectively use modern AI tools during development.
After generating the initial web application with AI-assisted tools, I tested the code, fixed issues, and made several customizations to improve the functionality and user experience.

# FEATURES
- Play the classic Snake Water Gun game against a computer opponent with randomly generated moves. 
- Track wins, losses, and draws throughout the game.
- View your current winning streak and best streak record.
- Unlock achievements by reaching different milestones.
- Keep a record of recent matches and results.
- Save game progress using local storage.
- Switch between dark and light themes based on preference.
- Sound effects for different game actions.
- Experience victory animations when winning a match.
- Use keyboard shortcuts for quicker gameplay.
- Enter a custom player name for a personalized experience.
- Start a new game or reset statistics at any time.
- Responsive design that works across different screen sizes.
- Clean and modern user interface.
